# Nutrition Information RAG System

A Retrieval-Augmented Generation (RAG) application that delivers accurate and context-aware nutritional information using semantic search and Large Language Models (LLMs).

---

## Features
- Semantic search for nutrition-related queries
- Embedding-based document retrieval
- Context-aware response generation
- Efficient nutritional data lookup
- Scalable data ingestion pipeline

---

## Tech Stack
- Python
- Pandas
- Sentence Transformers
- NumPy
- LLM Integration

---

## Project Workflow

1. Load and preprocess nutritional datasets
2. Split data into text chunks
3. Generate embeddings for chunks
4. Store embeddings for retrieval
5. Convert user query into embeddings
6. Retrieve relevant chunks using similarity search
7. Generate final response using LLM

---

## Example Queries

- "How often should infants be breastfed?",
- "What are symptoms of pellagra?",
- "How does saliva help with digestion?",
- "What is the RDI for protein per day?",
- "water soluble vitamins"
